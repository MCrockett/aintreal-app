# App Access Instructions for Reviewers

## Login Required
No - The app supports guest mode. Reviewers can play without creating an account or signing in.

---

## Testing Instructions

AIn't Real is a "Spot the AI" game where players identify which of two images is AI-generated.

### HOW TO TEST

#### 1. SOLO PLAY (No login required)
- Launch app → Select "Classic Solo" or "Marathon"
- Enter any name (e.g., "Reviewer")
- Tap "Create Game" to start
- Tap the image you think is AI-generated
- Complete the game to see results

#### 2. MULTIPLAYER (Requires 2 devices)
- Device 1: Select "Party Mode" → Create Game → Note the 4-letter code
- Device 2: Tap "Join Game" → Enter the code
- Host taps "Start Game" when ready

### RECOMMENDED TEST FLOW
1. Play a Classic Solo game (5 rounds) to test core gameplay
2. Try Marathon mode to test the streak/elimination mechanic
3. Optionally test Google Sign-In from the profile screen

### NOTES
- Test ads are displayed (AdMob test mode)
- All game modes work offline after initial load
- No in-app purchases

---

## Account Testing (Optional)

The app works fully in Guest mode.

For authenticated features (Google Sign-In):
- Reviewers can use their own Google account
- Or skip sign-in entirely - all game features work as guest

**No test credentials needed.**

---

## Account Deletion

Users can delete their account and data:
- In-app: Profile screen → "Delete Account" button
- Web: https://aint-real.com/delete-account
